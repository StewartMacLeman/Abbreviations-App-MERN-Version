# Software Development Abbreviations App.

This is a simple CRUD app to contain abbreviations that a user would wish to remember.

## Built with:
* The MERN stack
* CSS
---
## Seed data.

Add some example abbreviations with the below terminal command.

```
npm run seeds
```
